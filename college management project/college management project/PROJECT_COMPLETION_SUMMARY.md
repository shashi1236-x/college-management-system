## 🔧 CUSTOMIZATION OPTIONS

### Change Server Port
Edit `Constants.java`:
```java
public static final int SERVER_PORT = 9000;
```

### Switch Storage Mode
Edit `Constants.java`:
```java
USE_LOCAL_STORAGE = true;  // Local storage
USE_MYSQL = false;         // or MySQL
```

### Update Database
Edit `MySQLConnection.java`:
```java
DB_URL = "jdbc:mysql://your-host:3306/your-db";
DB_USER = "your-user";
DB_PASSWORD = "your-password";
```

### Modify Styling
Edit CSS files in `frontend/css/`:
- Colors, fonts, layouts
- All responsive design here

### Add New Features
Extend backend classes:
- Add new models in `models/`
- Create services in `services/`
- Create controllers in `controllers/`
- Add endpoints in `CollegeManagementServer.java`
